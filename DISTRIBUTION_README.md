# MarkNResoniteHeadlessController

ResoniteヘッドレスサーバーをブラウザUIから管理するためのツールです。

## 使用方法

### 初回実行

1. **MarkNResoniteHeadlessController.exe**をダブルクリックして実行
2. 初回実行時に必要な設定ファイルが自動生成されます
3. ブラウザで **http://localhost:8080** にアクセス
4. デフォルトパスワード **admin123** でログイン
5. 必要に応じてパスワードを変更してください

### 通常実行

1. **MarkNResoniteHeadlessController.exe**をダブルクリック
2. ブラウザで **http://localhost:8080** にアクセス
3. 設定したパスワードでログイン

## 設定ファイル

初回実行時に以下の設定ファイルが自動生成されます：

- `config/auth.json` - 認証設定
- `config/security.json` - セキュリティ設定
- `config/runtime-state.json` - ランタイム状態
- `config/restart.json` - 再起動設定
- `config/restart-status.json` - 再起動ステータス

## トラブルシューティング

### アプリケーションが起動しない

- **原因**: ポート8080が使用中
- **解決**: 他のアプリケーションを終了するか、環境変数`PORT`でポートを変更

### Web UIにアクセスできない

- **原因**: ファイアウォールの設定
- **解決**: ポート8080をファイアウォールで許可

### 設定ファイルが読み込めない

- **原因**: 設定ファイルの破損
- **解決**: 該当する設定ファイルを削除して再起動（自動再生成される）

## アップデート

1. 現在のアプリケーションを停止
2. 新しいexeファイルをダウンロード
3. 既存のexeファイルを置き換え
4. アプリケーションを再起動

**注意**: 設定ファイルは保持されます。
